# RPG Save System

一個使用 Python 製作的命令列 RPG 遊玩與存檔系統小專案。

本專案主要練習：

* Python dictionary 資料結構
* 函式設計
* 回合制戰鬥流程
* random 隨機事件
* pickle 存檔與讀檔
* 遊戲狀態持久化

## Features

目前已完成：

* 顯示主選單
* 顯示玩家狀態
* 建立玩家、怪物、道具等基礎資料
* 怪物資料支援防禦力 `defense`
* 玩家可以進入戰鬥
* 戰鬥中可以顯示敵人名稱與目前血量
* 支援玩家攻擊怪物
* 攻擊傷害會根據玩家攻擊力與怪物防禦力計算
* 傷害無效時會顯示 MISS
* MISS 時會隨機顯示彩蛋文字
* 支援暴擊系統
* 怪物血量歸零後，戰鬥會正確結束
* 擊倒怪物後可以獲得經驗值
* 支援 pickle 存檔
* 支援 pickle 讀檔
* 測試過關閉遊戲後重新讀檔，玩家經驗值可以成功保留
